## Backend for the calorie and macro tracking app

### Installation

You will need to install [node](https://nodejs.org/en/download/)

After that,

```sh
npm install
```

### Compile and hot-reload for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
